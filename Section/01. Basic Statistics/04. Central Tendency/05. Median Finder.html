বিজোড় সংখ্যক মান

Median = x(n+1)/2

যখন n বিজোড়, সরাসরি মধ্যবর্তী মান নিন।

উদাহরণ:

অবস্থান: (৫+১)/২ = ৩

মধ্যমা = ৩০

জোড় সংখ্যক মান

Median = (xn/2 + x(n/2)+1) / 2

যখন n জোড়, দুটি মধ্যবর্তী মানের গড় নিন।

উদাহরণ:

ডেটা: [১০, ২০, ৩০, ৪০]

n = ৪ (জোড়)

অবস্থান: n/২ = ২, (n/২)+১ = ৩

মধ্যমা = (২০ + ৩০) / ২ = ২৫
