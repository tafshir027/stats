সূত্রের ব্যাখ্যা:

বাইনোমিয়াল ডিস্ট্রিবিউশনের আশিত মান (গড়) সহজভাবে ট্রায়াল সংখ্যা এবং প্রতিটি ট্রায়ালে সাফল্যের সম্ভাবনার গুণফল। এটি সহজবোধ্য কারণ যদি আপনি একটি পরীক্ষা n বার পুনরাবৃত্তি করেন, এবং প্রতিবারের সাফল্যের সম্ভাবনা p হয়, তাহলে গড়ে প্রায় n × p বার সাফল্য আশা করতে পারেন।

n = 20, p = 0.7
E(X) = 20 × 0.7 = 14

একটি কারখানা ৯৫% গুণমান হার সহ আইটেম উৎপাদন করে। ২০০টি আইটেমের ব্যাচে, কতটি পরিদর্শনে উত্তীর্ণ হবে বলে আশা করা যায়?

n = 200, p = 0.95
E(X) = 200 × 0.95 = 190

ধাপ ১: আশিত মান সংজ্ঞায়িত করুন

একটি বিচ্ছিন্ন র্যান্ডম ভেরিয়েবল X-এর আশিত মান সংজ্ঞায়িত করা হয়:

E(X) = Σ [x × P(X = x)]
২

ধাপ ২: বাইনোমিয়াল ডিস্ট্রিবিউশনে প্রয়োগ করুন

প্যারামিটার n এবং p সহ বাইনোমিয়াল ডিস্ট্রিবিউশনের জন্য:

E(X) = Σk=0n [k × C(n,k) × pk × (1-p)n-k]
৩

ধাপ ৩: সমষ্টি সরলীকরণ করুন

সমাবেশবিধি পরিচয় এবং বীজগাণিতিক হেরফের ব্যবহার করে:

E(X) = n × p
৪

ধাপ ৪: সহজবোধ্য বোঝাপড়া

ফলাফলটি সহজবোধ্য: প্রতিটি ট্রায়াল আশিত মানে p অবদান রাখে, এবং n স্বাধীন ট্রায়াল সহ, মোট আশিত মান n × p।
