ভরযুক্ত গড় সূত্রসমূহ

ভরযুক্ত গড় সূত্র

x̄_w =
Σ(w_i × x_i)
/ Σw_i

যেখানে:

  • x̄_w = ভরযুক্ত গড়
  • w_i = i-তম মানের ভর
  • x_i = i-তম মান
  • Σ = যোগফল (মোট)

এক্সটেন্ডেড সূত্র

x̄_w =
w₁x₁ + w₂x₂ + ... + wₙxₙ
/ (w₁ + w₂ + ... + wₙ)

বিকল্প নোটেশন:

  • w₁, w₂, ..., wₙ = স্বতন্ত্র ভর
  • x₁, x₂, ..., xₙ = স্বতন্ত্র মান
  • n = ডেটা পয়েন্টের সংখ্যা

কখন ভরযুক্ত গড় ব্যবহার করবেন

  • জিপিএ গণনা (ভর হিসাবে ক্রেডিট)
  • কোর্স গ্রেড (ভর হিসাবে কঠিনতা)
  • বিনিয়োগ রিটার্ন (ভর হিসাবে বিনিয়োগকৃত পরিমাণ)
  • সার্ভে ফলাফল (ভর হিসাবে উত্তরদাতার গুরুত্ব)
  • পরিসংখ্যানে ভরযুক্ত গড়
