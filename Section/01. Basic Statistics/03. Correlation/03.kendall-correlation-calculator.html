কেন্ডাল টাউ-বি

বর্গাকার কন্টিনজেন্সি টেবিলের জন্য, উভয় ভেরিয়েবলে টাইসের জন্য সমন্বয় করে

τb = (C - D) / √[(C + D + Tₓ)(C + D + Tᵧ)]
  • C = কনকর্ডেন্ট পেয়ারের সংখ্যা
  • D = ডিসকর্ডেন্ট পেয়ারের সংখ্যা
  • Tₓ = শুধুমাত্র X-এ টাই সংখ্যা
  • Tᵧ = শুধুমাত্র Y-এ টাই সংখ্যা
  • পরিসর: -১ থেকে +১

কেন্ডাল টাউ-সি

আয়তক্ষেত্রাকার টেবিলের জন্য, টেবিলের মাত্রার জন্য সমন্বয় করে

τc = (C - D) / [½n²(m-1)/m]
  • C = কনকর্ডেন্ট পেয়ারের সংখ্যা
  • D = ডিসকর্ডেন্ট পেয়ারের সংখ্যা
  • n = পর্যবেক্ষণের সংখ্যা
  • m = ন্যূনতম (সারি, কলাম)
  • পরিসর: -১ থেকে +১

কেন্ডাল টাউ বুঝুন

কেন্ডাল টাউ দুটি পরিমাপিত পরিমাণের মধ্যে অর্ডিনাল অ্যাসোসিয়েশন পরিমাপ করে। এটি একটি নন-প্যারামেট্রিক পরিসংখ্যান যা প্রতিটি পরিমাণ দ্বারা র্যাঙ্ক করা হলে ডেটার ক্রমের সাদৃশ্য মূল্যায়ন করে। পিয়ারসনের পারস্পরিক সম্পর্কের বিপরীতে, এটি রৈখিকতা ধরে নেয় না এবং আউটলিয়ারগুলির জন্য আরও শক্তিশালী।

টাউ-বি বনাম টাউ-সি তুলনা

কখন কোন টাউ ব্যবহার করবেন?

টাউ-বি পছন্দনীয় যখন:
  • বর্গাকার কন্টিনজেন্সি টেবিল
  • উভয় ভেরিয়েবলে টাই আছে
  • অনেক টাই সহ অর্ডিনাল ডেটা
  • পরিসর: -১ থেকে +১
টাউ-সি পছন্দনীয় যখন:
  • আয়তক্ষেত্রাকার টেবিল
  • বিভিন্ন বিভাগ গণনা
  • বড় নমুনা আকার
  • পরিসর: -১ থেকে +১
দ্রষ্টব্য: টাইস থাকলে টাউ-বি ±১ এ পৌঁছাতে পারে না, যখন টাইস থাকলেও টাউ-সি ±১ এ পৌঁছাতে পারে।
