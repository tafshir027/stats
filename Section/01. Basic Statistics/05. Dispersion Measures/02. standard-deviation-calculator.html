সূত্র এবং সংজ্ঞা

স্যাম্পল স্ট্যান্ডার্ড ডেভিয়েশন

নমুনায় বিস্তার পরিমাপ করে (পপুলেশনের উপসেট)

s = √[∑(xᵢ - μ)²/(n - 1)]
  • s = নমুনা স্ট্যান্ডার্ড ডেভিয়েশন
  • xᵢ = প্রতিটি পৃথক মান
  • μ = নমুনার গড়
  • n = নমুনার আকার (নিরপেক্ষ অনুমানের জন্য n-1 ব্যবহার)

পপুলেশন স্ট্যান্ডার্ড ডেভিয়েশন

সম্পূর্ণ পপুলেশনে বিস্তার পরিমাপ করে

σ = √[∑(xᵢ - μ)²/N]
  • σ = পপুলেশন স্ট্যান্ডার্ড ডেভিয়েশন
  • xᵢ = প্রতিটি পৃথক মান
  • μ = পপুলেশনের গড়
  • N = পপুলেশনের আকার (N ব্যবহার, N-1 নয়)

ভ্যারিয়েন্স

গড় থেকে বর্গ পার্থক্যের গড়

σ² = ∑(xᵢ - μ)²/N
  • σ² = ভ্যারিয়েন্স (সিগমা স্কোয়ার্ড)
  • স্ট্যান্ডার্ড ডেভিয়েশনের বর্গ
  • নমুনা ভ্যারিয়েন্স হর হিসেবে (n-1) ব্যবহার
  • পপুলেশন ভ্যারিয়েন্স হর হিসেবে N ব্যবহার
